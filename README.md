# Weather App Assignment

## Description

Weather App with React.js and api.met.no/weatherapi

## Installation

To install

`npm install`

To run

`npm start`

`npm run server`

## To build

`npm run build`

Then use the dist folder you get.

## Built With

- React.js
- Scss
- api.met.no/weatherapi
